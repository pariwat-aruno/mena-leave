เริ่มใช้งานครั้งแรก
  1. กดเมนู "ส่งใบลา" หรือ "ใบลาของฉัน" ที่ rich menu ด้านล่าง
  2. ระบบจะให้คุณลงทะเบียน — กรอกชื่อ-นามสกุล รหัสพนักงาน แผนก ตำแหน่ง
  3. กดส่ง → รอ HR อนุมัติ (ระบบจะส่งข้อความแจ้งคุณผ่าน LINE)
  4. เมื่อ HR อนุมัติ → คุณส่งใบลาได้ทันที
วิธีส่งใบลา
  1. กดเมนู "ส่งใบลา"
  2. ดูโควตาคงเหลือ + เงื่อนไขการลาของแต่ละประเภท
  3. เลือกประเภท → เลือกวันที่เริ่ม-สิ้นสุด → กรอกเหตุผล
  4. แนบไฟล์ภาพ (ถ้ามี) เช่น ใบรับรองแพทย์ บัตรคิวคลินิก
  5. กด "ถัดไป" → อนุญาตให้ระบบเข้าถึง GPS ตอนนี้
  6. ตรวจสรุป → กด "ส่งใบลา"
  7. รอผู้อนุมัติ (ระบบส่งข้อความแจ้งคุณทุกขั้น)
GPS ใช้ตอนกดส่งเท่านั้น — ไม่ติดตามตำแหน่งคุณตลอดเวลา
ลำดับการอนุมัติ 3 ชั้น

ทุกใบลาผ่าน 3 ชั้น:

  1. ชั้น 1 หัวหน้างาน — ตรวจเหตุผลของลูกน้องตัวเอง
  2. ชั้น 2 ฝ่ายบุคคล (HR) — ตรวจโควตา + กฎ
  3. ชั้น 3 ผู้บริหาร — อนุมัติชั้นสุดท้าย

กรณีพิเศษ: ถ้าคุณเป็นหัวหน้างานเอง → ระบบข้ามชั้น 1 ตรงไปชั้น 2
ถ้าผู้ลาเป็น HR เอง → ข้ามชั้น 1+2 ตรงไปชั้น 3

ดูประวัติใบลาของฉัน
  1. กดเมนู "ใบลาของฉัน"
  2. เห็นใบลาทุกใบ พร้อมสถานะ
  3. แตะแต่ละใบ เพื่อดูรายละเอียด + ลำดับการอนุมัติ
ลาย้อนหลัง (ลาป่วยฉุกเฉิน)

หากเจ็บป่วยฉุกเฉินจนต้องลาก่อนแจ้ง:

  1. กลับมาทำงานแล้วเปิดเมนู "ส่งใบลา"
  2. เลือกประเภท "ลาป่วย" เท่านั้น (ลากิจ/พักร้อนย้อนหลังไม่ได้)
  3. เลือกวันที่ลาตามจริง (ระบบจะ mark เป็น "ย้อนหลัง")
  4. แนบใบรับรองแพทย์ถ้าลา 3 วันขึ้นไป
โควตาคงเหลือไม่พอ ทำยังไง?
