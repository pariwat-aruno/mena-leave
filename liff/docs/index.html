4 บทบาท (Roles):
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
  │ VISITOR  │ →  │   USER   │ →  │  ADMIN   │ →  │  OWNER   │
  │ ยังไม่    │    │ พนักงาน  │    │ ฝ่ายบุคคล │    │ ผู้บริหาร   │
  │ ลงทะเบียน│    │+หัวหน้างาน│    │   (HR)   │    │          │
  └──────────┘    └──────────┘    └──────────┘    └──────────┘

3 ชั้นอนุมัติใบลา (3-stage approval):
  ผู้ลาส่งใบลา
       ↓
  Stage 1: หัวหน้างาน (USER + is_supervisor=TRUE)
       ↓
  Stage 2: ฝ่ายบุคคล (ADMIN) — คนใดคนหนึ่งกดได้ (1st wins)
       ↓
  Stage 3: ผู้บริหาร (OWNER) — คนใดคนหนึ่งกดได้
       ↓
  ✅ อนุมัติแล้ว (final_status=approved + หักโควตา)

ข้อยกเว้น:
  • หัวหน้างานลาเอง → ข้าม Stage 1 → เริ่มที่ Stage 2
  • ADMIN ลาเอง → ข้าม Stage 1+2 → ตรงไป Stage 3
  • OWNER ลาเอง → auto-approve (มี audit log)
